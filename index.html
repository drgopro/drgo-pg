닥터고블린컴퍼니 CRM

방송 세팅 및 고객 관리 프로세스 최적화를 위한 시스템 화면 및 기능 명세 가이드

UI 와이어프레임 프론트/백엔드 요구사항 통합 커뮤니케이션 문서
시스템 아키텍처
CRM 백오피스 코어를 중심으로 3개 핵심 모듈과 외부 API가 연결됩니다.

고객/이력 관리

  • 의뢰인 DB 구축 및 세부 정보 관리
  • 상담 및 방문 이력 추적
  • 다중 이력을 '프로젝트'로 그룹화

자산/세팅 모듈

  • 방송 장비 공간 시각화 맵핑
  • 소프트웨어 계층 구조(Depth) 설정
  • 작업 항목별 소요 시간 및 견적 자동 계산

외부 연동 [API]

  • 구글 캘린더 양방향 일정 동기화
  • 카카오 알림톡 48시간 전 리마인드
  • 개인 계정 기반 로그인 및 권한 관리

📋 [1] 의뢰인 기본 정보 입력 및 수정▼
플랫폼

플랫폼/콘텐츠 다중 연동

유튜브, 트위치, 치지직 등 다중 체크박스 구조. DB 저장 시 배열(Array)로 관리하여 확장성 확보.

등급

등급 분류 라벨링

일반 / VIP / 렌탈 드롭다운 선택. 고객 중요도에 따른 시각적 라벨링 UI 동기화.

이력

수정 이력 시인성 확보

기존 데이터 수정 시, 변경 내용을 다른 색상으로 UI 하이라이트 처리 (미작성/회색 텍스트와 명확히 구분).

증빙

증빙 서류 업로드

현금영수증, 사업자등록증 등 다중 이미지 파일 업로드. 용량 및 확장자 체크 방어 로직 필수.

🔍 [2] 통합 이력 조회 (상담 vs 프로젝트 의뢰)▼
플랫

기본 상담 뷰: 플랫 리스트

최신 상담 순서대로 나열. '중요' 체크박스 선택 항목은 별표(*) 아이콘으로 필터링 시각화.

계층

프로젝트 의뢰 뷰: 계층형

1의뢰 = 1프로젝트. 상위 리스트 클릭 시 아코디언으로 하위 이력(상담, 결제, 방문) 일괄 전개.

✏️ [3] 상담 이력 작성 폼 및 동적 분기▼
자동화

필드 동기화

닉네임, 상담일(당일) 자동 세팅. 상담사는 로그인 계정 기준으로 자동 입력, 권한에 따라 수정 가능.

상태

상태 값 라이브 연동

상담결과 드롭다운(진행중, 대기, 완료 등) 선택 시, 의뢰인 리스트의 마스터 상태 라벨과 즉시 동기화.

분기

결제/예약 폼 분기 처리 (Toggle UI)

'결제', '예약' 체크박스 클릭 시 → 페이지 이동 없이 현재 창 하단에 결제/예약 입력 폼이 즉시 확장(Toggle). 업무 흐름 단절 완벽 차단.

🔧 [4] 방문 보고서 및 세팅 명세▼
체크

직관적 완수 표시 (Strikethrough)

현장 기사가 세부 작업(PC 조립, 모니터 설치 등) 체크 시, 해당 텍스트에 '취소선' 발생. 모바일에서도 진척도를 오판 없이 파악.

추가

현장 추가 결제 및 예약 인라인 연동

돌발적 추가 구매나 재방문 일정 발생 시, '추가결제', '재방문 예약' 체크로 별도 메뉴 이동 없이 즉석 처리.

🖥️ [5] 방송 장비 공간 맵핑 및 자동 계산▼
공간

공간 직관성 아이콘 맵핑

데스크, PC, 모니터, 마이크의 실제 위치를 아이콘 기반 그래픽으로 시각화. 담당자 교체 시에도 세팅 상황 즉각 파악.

SW

소프트웨어 계층화 (Drag & Drop)

XSplit > 프릭샷 등 방송 프로그램 간의 종속 관계를 직관적 Drag & Drop으로 설정. 최대 5단계 Depth 계층 지원.

조명

조명 디테일 컨트롤러

최대 8개 개별 조명의 정면(정)/반사면(반) 조향 각도를 라디오 버튼으로 제어, 구체적 제품명 입력 폼.

계산

백그라운드 자동 계산 엔진

특정 기술 항목 체크 시 → 단가표 매핑으로 전체 예상 소요 시간 및 기본 청구 금액이 실시간 자동 합산.

💳 [7] 결제 내역 및 캘린더 예약 동기화▼
견적

이전 견적 연동 시스템

상담 단계에서 저장된 견적서를 원클릭으로 불러와 결제 폼에 자동 입력.

분리

자산 및 서비스 명세 분리 UI

물리적 자산 [제품]과 무형의 [서비스]를 좌우/상하로 명확히 분리 표기하여 투명한 최종 결제 금액 산출.

캘린더

구글 캘린더 예약 동기화 엔진

방문 기술자 할당 + 예약 날짜/시간 지정 완료 시 → DB 등록과 동시에 구글 캘린더 API를 백그라운드로 호출하여 사내 공용 캘린더에 즉시 일정 자동 생성.

시스템 자동화 및 외부 API
사람의 개입 없이 자동으로 처리되는 3가지 핵심 프로세스입니다.

💬 카카오 알림톡 리마인드

예약 확정 DB 저장 ↓ 48시간 전 타이머 도래 ↓ 스케줄러 자동 인식 ↓ 의뢰인에게 알림톡 발송

📅 캘린더 태깅 룰

'임시/제안' 체크 활성화 ↓ 구글 캘린더 API 전송 ↓ 일정 제목에 [임시] 또는 [제안] 태그 강제 접두사 생성

🔄 AS 만료 자동 처리

상태값: 방문 완료 ↓ 14일(2주) 타이머 작동 ↓ 연장 이력 조회 (특이사항 없음) ↓ 프로젝트 의뢰 강제 종료
대시보드 위젯
로그인 후 메인 화면에서 바로 확인할 수 있는 4가지 핵심 위젯입니다.
📅

미니 캘린더 현황판

월간 미니 현황판으로 일정 제목, 진행 구분 색상, 지역 데이터를 압축 시각화. 상담원의 빠른 일정 파악 지원.

📝

지식 공유 게시판

공지사항, 회의록, 최근 이슈, 기술 정보 아카이빙. 상담사 이상 권한 전용 작성/조회. 클릭 시 상세 페이지 이동.

🌐

외부 VOC 연동 뷰

회사 홈페이지 QNA 게시판 최신 글을 대시보드 내에 연동. 작성자 닉네임, 등록 날짜, 제목 리스트화.

💬

아웃바운드 요약 데이터

외부 채팅 솔루션 연동. 아웃바운드 텍스트 내용을 제목 위주로 리스트화하여 CRM 이력으로 흡수 및 추적.

🔐 개인 계정 로그인

1
계정 발급 — 관리자(소유자)가 각 사원에게 아이디와 임시 비밀번호를 발급합니다.
2
첫 로그인 — 발급받은 계정으로 로그인 후 비밀번호를 변경합니다.
3
역할 배정 — 소유자·관리자·상담사·외부(기술자) 4단계 역할이 자동 부여됩니다.
본 CRM 백오피스의 진정한 핵심 가치는 단순한 고객 데이터의 저장이 아닙니다.
직관적인 UI를 통해 '상담 및 세팅 작업 시간을 극적으로 단축'하고, 흩어진 정보를 하나의 프로젝트로 묶어 '투명한 작업 이력 트래킹'을 달성하는 것에 있습니다.
