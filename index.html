업무 흐름
고객 문의부터 서비스 완료까지 이렇게 진행됩니다.
1

고객 문의 접수

전화, 채팅, 이메일 등으로 고객(크리에이터/스트리머) 문의가 들어옵니다.

2

고객 등록 및 일정 생성

신규 고객이면 CRM에 등록하고, 방문 일정(날짜·시간·담당자)을 잡습니다.

3

현장 방문 및 서비스

방문 유형(설치, A/S, 상담, 컨설팅)에 따라 현장 서비스를 진행합니다.

4

결과 기록 및 후속 조치

방문 결과를 메모하고, 필요 시 추가 장비 주문이나 배송을 연결합니다.

5

주문 · 배송 처리

필요 장비를 주문 처리하고 배송 상태를 추적합니다. 모든 이력이 고객 카드에 남습니다.

모듈 상세 안내
각 모듈을 클릭하면 상세 내용을 확인할 수 있습니다.
📅 방문 일정 관리▼
기본 정보 일정 제목, 방문 날짜, 시작/종료 시간, 방문 장소, 상세 설명
분류 · 상태 유형(방문·설치·A/S·상담), 상태(예정·완료·취소), 담당 직원
캘린더 뷰 월간·주간·일간 보기, 담당자별 필터링, 드래그로 일정 이동
결과 기록 방문 후 현장 상황, 추가 필요 장비, 후속 조치 메모 저장
👤 고객 정보 · 이력 관리▼
고객 기본 정보 활동명, 실명, 연락처, 이메일, 플랫폼(트위치·유튜브·치지직 등), 채널 주소
고객 상태 관리 리드(잠재)·활성·비활성 분류, 담당 직원 배정
상담 이력 전화·방문·채팅·이메일·메모별 상담 내용 기록 및 시간순 조회
통합 히스토리 방문 일정, 주문, 배송 내역이 고객 카드에 자동 연결

제품 등록 카테고리 분류, 상품명, 상품 코드(SKU), 매입가/판매가, 상세 설명
재고 현황 현재 수량 실시간 확인, 안전 재고 미달 시 알림
입출고 기록 입고·출고·조정·반품 유형별 수량 변동 이력 관리
상품 상태 판매중·단종·품절 상태 관리, 주문과 자동 연동
🚚 배송 관리▼
주문 등록 고객·상품 선택, 수량 입력, 주문번호 자동 생성, 견적서 겸용
주문 상태 초안·확정·진행중·완료·취소 단계별 관리
배송 추적 택배사·운송장 번호 입력, 배송 상태(준비·발송·완료) 추적
금액 관리 총액·할인·최종 결제 금액 자동 계산, 주문 이력 조회

🔐 개인 계정 로그인

1
계정 발급
관리자가 각 사원에게 아이디(이메일)와 임시 비밀번호를 발급합니다.
2
첫 로그인
발급받은 계정으로 로그인 후 비밀번호를 변경합니다.
3
역할별 권한
관리자(전체 관리), 매니저(팀 관리), 일반 사원(본인 업무) 3단계로 나뉩니다.

🛡️ 역할별 접근 권한

관리자 (Admin) 사원 계정 관리, 전체 데이터 조회·수정·삭제, 시스템 설정
매니저 (Manager) 팀 일정·고객 관리, 재고 입출고 승인, 주문 확정
일반 사원 (Staff) 본인 담당 일정·고객 조회, 상담 기록 작성, 재고 확인
공통 캘린더 조회, 고객 검색, 제품 목록 확인
